# News App
This is an android application made using Java. It shows top 10 Headlines.

## Features
- Swipe Down to Refresh
- Dark Mode
- 5 Countries
- 5 Languages
- Autosave settings

## Implements
- Volley Library
- RecyclerView
- SwipeRefreshLayout
- SharedPreferences
- AlertFragment
